# Welcome to my blog!

I made it in a single day because production comes first, features come later. I think that scaling it with blog posts and adding features once they're relevant is more important than just making the whole thing as perfect as possible. 

Tech Stack:
- [SolidStart](https://start.solidjs.com/) (web framework)
- [Pico.css](https://picocss.com/) (css framework)
- [mdx](https://mdxjs.com/) (markdown)
- [Vercel](https://vercel.com/) (host)
